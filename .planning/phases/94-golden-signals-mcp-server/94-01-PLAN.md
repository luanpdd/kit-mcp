# Phase 94 Plan 01: Golden Signals MCP Server

## Objective

Apply the `four-golden-signals` skill to kit-mcp itself. Add an in-memory metrics module (counters + per-tool latency histogram), wrap the MCP central catch with instrumentation, and expose readings via a new `metrics-snapshot` tool. Zero new dependencies.
